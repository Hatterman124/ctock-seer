struct.h defines common objects (such as the database) and constants (such as UL_LOW, the lowest value an unsigned int can have).

main.cpp is called first. It repeatetly calls menu() in menu.h/cpp until quit.
From menu(), user typically calls python_get_df() from util.h/cpp to download ticker dataframe using yfinance.
Back to the menu, the user can call print.h/cpp to print information about the dataframe they downloaded.
They can also start the lstm by calling lstm.h/cpp.
